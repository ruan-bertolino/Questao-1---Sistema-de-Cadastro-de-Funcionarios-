# Trabalho Java - Sistema de Funcionários

Este projeto é um exercício acadêmico em Java que implementa um sistema simples de gerenciamento de funcionários e departamentos.

## Estrutura do Projeto
- **src/**: contém as classes Java principais (`Departamento.java`, `Funcionario.java`, `TesteSistema.java`).
- **SistemaFuncionarios/**: pasta com os arquivos de configuração do Eclipse.
- **.gitignore**: arquivo de configuração para ignorar arquivos desnecessários no repositório.

## Como executar
1. Importe o projeto no Eclipse ou outra IDE Java.
2. Compile e execute a classe `TesteSistema.java`.
3. O programa irá demonstrar o funcionamento básico do sistema de funcionários.

## Objetivo
O objetivo deste trabalho é praticar conceitos de:
- Criação de classes e objetos em Java
- Organização de projetos em IDEs
- Uso do Git e GitHub com `.gitignore` para manter o repositório limpo
